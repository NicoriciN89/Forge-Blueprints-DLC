# Forge Blueprints DLC

![image](https://github.com/Jods-Its/Forge-Blueprints-DLC/blob/main/IMGs/FBCover.png)

> A mod for The Long Dark, by Hinterland Studios, that add some blueprints to the forge.

## Disclaimer

This mod is an updated version of ds5678's [Forge Blueprints Mod](https://github.com/ds5678/ForgeBlueprintsMod). The only new thing is updated crafting icons.

## Usage 

![image](https://github.com/Jods-Its/Forge-Blueprints-DLC/blob/main/IMGs/20230615154843_1.jpg)

> Go to a forge, where you can now craft essential tools and equipment:
- **Tools**: Cooking Pots, Prybars, Hammers, Hatchets
- **Knives**: Improvised Knife, Simple Tools Knife  
- **Arrow Production**: Efficient arrow heads (7 per batch), recycle arrow heads for scrap
- All recipes improve your **Gunsmithing skill**!

## Installation 

* Download latest .modcomponent file in [releases](), unzip, drop both .modcomponent and .dll files in the mod folder and enjoy!
* Requires [ModComponent](https://github.com/dommrogers/ModComponent) and all its dependencies to work.

## Contents

**V.2.0.0 (2026 Update):**
* **Major balancing pass** - reduced resource requirements for better gameplay flow
* **Skill integration** - all recipes now improve Gunsmithing skill
* **Full localization** - supports all TLD languages (18 languages total):
  - English, Russian, German, French, Japanese, Korean
  - Chinese (Simplified & Traditional), Swedish, Turkish
  - Norwegian, Spanish, Portuguese (PT & BR), Dutch
  - Finnish, Italian, Polish, Ukrainian
* **New blueprints added:**
  - Improvised Knife (3 scrap metal, 180 min)
  - Hatchet (6 scrap metal + 1 hardwood, 270 min)
  - Simple Tools Knife (2 scrap metal, 150 min)
* **Resource rebalancing:**
  - Cooking Pot: 6→5 scrap metal
  - Hammer: 10→8 scrap metal
  - Prybar: 6→5 scrap metal

**V.1.6.0 (2023):**
* Updated to post DLC TLD.
* New crafting icons.

**V.1.5.0 (Pre-DLC):**
* Craft cooking pots, prybars and hammers in the forge.
* Craft arrow head in batches or recycle them for scrap metal.
